# Medical Appointment Booking

This is the frontend capstone project for the **Medical Appointment Booking** website. This application provides a platform for patients to discover doctors and book appointments, while allowing healthcare providers to manage their schedules.

## Project Overview
The goal of this project is to create a seamless user experience for healthcare management using **React**.
